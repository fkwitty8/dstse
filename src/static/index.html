Model Notes

The Bayesian Network (BN) core is static and not updated by feedback — it encodes structured domain knowledge about Kampala weather patterns, road conditions, and lecture disruption probabilities. Feedback from lecturers is used to fine-tune the supervised classification layer, improving edge-case predictions over time. Both layers complement each other for more robust decisions.
